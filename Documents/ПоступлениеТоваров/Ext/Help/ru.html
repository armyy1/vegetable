Документ предназначен для примера использования ряда подсистем.

В подсистеме Базовая функциональность документ показывает пример копирования и вставки строк табличной части Товары с помощью процедур
СкопироватьСтрокиВБуферОбмена, СтрокиИзБуфераОбмена общего модуля ОбщегоНазначения. Копирование также можно выполнять между разными документами, содержащими табличную часть Товары:
Демо: Оприходование товаров
Демо: Перемещение товаров
Демо: Реализация товаров
Демо: Списание товаров

Подсистема Варианты отчетов (совместно с подсистемой Подключаемые команды) добавляет в подменю Отчеты форм документа и списка демонстрационные команды для открытия контекстных отчетов Связанные документы и Места использования. См. ВариантыОтчетовПереопределяемый.ПередДобавлениемКомандОтчетов.

В подсистеме Версионирование при документах сохраняются их версии.

В подсистеме Даты запрета изменения документ показывает пример запрета изменения документа по двум разделам одновременно:

  • по разделу Демо: Торговля с объектом Демо: Группа доступа партнеров, который вычисляется "через две точки" от реквизита Контрагент;
  • по разделу Демо: Складской учет с объектом Демо: Место хранения, который извлекается из реквизита Место хранения.

Подсистема Отчет о движениях документа на примере этого документа демонстрирует возможность добавления в отчет произвольного набора данных (выводит записи регистра расчета Демо: Основные начисления сотрудников). См. общий модуль ОтчетОДвиженияхДокументаПереопределяемый.

В подсистеме Префиксация объектов к номеру документа добавляются префиксы организации и информационной базы.

В подсистеме Управление доступом документ показывает пример типичного использования логики ограничения по трем видам доступа.

  • Список ограничивается по трем видам доступа:
    • Организации (проверяется значение реквизита Организация);
    • Демо: Группы партнеров (проверяется значение реквизита Партнер);
    • Демо: Места хранения (проверяется значение реквизита Место хранения);
    • все значения должны быть разрешенными одновременно (в одной из групп доступа).
